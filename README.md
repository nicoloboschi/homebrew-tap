# Hindsight Homebrew tap

```bash
brew install --cask nicoloboschi/tap/hindsight
```

Installs the [Hindsight](https://github.com/nicoloboschi/hindsight-desktop)
menu-bar app and `uv`. The app is currently unsigned, so on first launch:
right-click the app → Open (or `xattr -dr com.apple.quarantine /Applications/Hindsight.app`).
